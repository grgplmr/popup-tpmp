# TPMP Popup

Plugin WordPress simple qui affiche une image dans une popup 2 secondes après le chargement du site. La popup inclut un bouton de fermeture bien visible et peut être fermée en cliquant en dehors de l'image ou via la touche Échap.

## Installation

1. Copiez le dossier `tpmp-popup` dans le répertoire `wp-content/plugins/` de votre site WordPress.
2. Activez le plugin « TPMP Popup » depuis l'interface d'administration WordPress.

## Personnalisation

- Remplacez l'image située dans `tpmp-popup/assets/popup-image.svg` par votre propre visuel si besoin.
- Les styles peuvent être ajustés dans `tpmp-popup/assets/popup.css`.
- Le délai d'affichage (2 secondes par défaut) est défini dans `tpmp-popup/assets/popup.js`.
